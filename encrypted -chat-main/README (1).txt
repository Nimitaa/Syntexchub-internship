# 🔐 Encrypted Client–Server Chat Application

## 📌 Overview

This project implements a secure client–server chat system in Python where messages are encrypted before being transmitted
across the network. It demonstrates how secure communication can be built using socket programming and symmetric encryption.

This project shows 
Advanced Encryption Protocol
Tramisssion control protocol
---

## 🚀 Features

* 🔒 Message encryption using AES
* 🌐 TCP socket communication
* 🔑 Pre-shared symmetric key
* 🔁 Random Initialization Vector (IV) for every message
* 👥 Multiple clients supported using threading
* 📝 Server-side message logging
* 💻 Terminal-based chat interface

---

## 🧠 Security Concepts Demonstrated

Symmetric encryption
Secure message transmission
Random IV generation
Client-server architecture
Concurrent client handling

Encryption is implemented using the **Advanced Encryption Standard (AES)** and communication 
occurs through **Transmission Control Protocol (TCP)** sockets.

## 🏗️ Project Architecture

```text
           +----------------+
           |     Server     |
           |   server.py    |
           +--------+-------+
                    |
        +-----------+-----------+
        |                       |
  +-----+------+         +------+-----+
  |   Client 1  |         |  Client 2 |
  |  client.py  |         | client.py |
  +-------------+         +------------+
```
**server.py**
Handles incoming connections, broadcasts messages, and logs encrypted messages.

**client.py**
Connects to the server, encrypts outgoing messages, and decrypts received messages.

**crypto_utils.py**
Contains encryption and decryption functions using AES.

**chat_log.txt**
Stores encrypted message logs generated by the server.

Each client:

1. Encrypts messages before sending
2. Sends encrypted data to the server
3. Receives encrypted messages from other clients
4. Decrypts messages for display
